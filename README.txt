# Task Tracker App

## Overview

The Task Tracker App is a simple React application that allows users to add, view, and delete tasks. It features a clean interface with a gradient background, a header, an input form for task entry, and a list to display tasks. This app demonstrates fundamental React concepts such as state management, props, and component interaction.

## Features

- **Add tasks**: Enter tasks into a form and add them to the list.
- **View tasks**: See all added tasks displayed in a list.
- **Delete tasks**: Remove tasks from the list individually.
- **Responsive design**: Features a gradient background and a clean user interface.

## Components

- **App**: Manages the state of the task list and handles task addition and deletion.
- **Header**: Displays a welcoming message at the top of the page.
- **TaskInput**: Provides an input form for users to enter new tasks.
- **TaskList**: Renders a list of tasks and includes a delete button for each task.
- **TaskDetails**: Displays individual task details and a delete button.
